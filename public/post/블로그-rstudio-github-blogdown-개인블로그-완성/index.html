[블로그] Rstudio + github + blogdown = 개인블로그 완성!

2018 Oct 10
R / Rmd / blogdown / github pages

세줄요약

  1. R로 보고 문서를 만드는 rmd로 블로그를 만들 수 있다.
  2. md 기반의 정적 사이트 생성기가 많지만, R 패키지로 제공되는 blogdown이 좋더라.
  3. 전에 소개한 Github Page로 https://<user_id>.github.io/blog 주소의 개인 블로그를 만들어 보자.

본 포스트는 영상으로 진행하였습니다.
